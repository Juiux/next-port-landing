Arch Installation Guide (UEFI only)
-------------------------------------

[INTRODUCTION]

This is a no bullshit tutorial to install Arch Linux on UEFI systems.
Disclaimer : I'm not responsible if your device tries to eat you or suddenly become a time bomb. Do this at your own risk.
Note : If you are on MBR, consider looking for another countless tutorials on internet or just refer to Arch Wiki.
You can read about Arch Linux here https://en.wikipedia.org/wiki/Arch_Linux.

[PREREQUISITES]

1. Arch Linux ISO. You can download it here.
2. A working Arch Linux Live Boot environment.
> You can make one by using Rufus.
>> Step-by-step :
>> 1. Make sure that you already downloaded Arch Linux ISO and Rufus. (Note: Links are underlined words above)
>> 2. Plug your installation medium into your device. (Note: A flashdisk with 8GB of storage would be enough)
>> 3. Run Rufus.
>> 4. Select your installation medium at the device section. (Note: This will erase anything in your flashdisk so make sure to back up any valuable contents before proceeding)
>> 5. Select the boot selection and find the Arch Linux ISO that you previously downloaded.
>> 6. Select the partition scheme. (I'm using GPT on this tutorial)
>> 7. Leave all other settings to default and hit that start button.
>> 8. If prompted to select method, use DD method.
>> 9. Wait for it to finish install.
